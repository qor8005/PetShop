### 1. 프로젝트 버전
* spring-boot : 2.7.5
* java : 17
* oracle : 11.2.0.2.0
* lombok : 1.18.24

### 2. 주요기능
* 회원가입 로그인 기능(artist, user의 2가지 타입이 있습니다.)
* 지정펀딩과 미지정 펀딩 2가지의 방식으로 펀딩 글을 작성합니다.
* 펀딩을 위한 결재 취소 환불 등의 기능이 있습니다.
* 알림(댓글작성, 펀딩 달성, 마감, 취소 등의 상황에서 알림이 발송됩니다)
* 유저, 아티스트의 개인 페이지(아티스트는 자신의 소개 페이지를 작성합니다)
